# Administration

Welcome to the Administration learning path! Here you'll learn system administration skills.

## Topics

| Topic | Description |
|:------|:------------|
| [Devices](devices/) | Understand device management and the /dev directory |
| [Filesystem](filesystem/) | Learn about filesystem hierarchy, types, and management |
| [Boot System](boot-system/) | Understand the Linux boot process |
| [Kernel](kernel/) | Learn about the Linux kernel |
| [Init](init/) | Understand init systems and service management |
| [Process Utilization](process-utilization/) | Monitor and manage system resources |
| [Logging](logging/) | Work with system logs |

These topics build on the Fundamentals path. Make sure you're comfortable with command line basics before diving in.
