# Devices Quiz Answers

## 1. /dev directory

**Question:** Where are device files stored on the system? (Please provide the absolute path. The answer is case-sensitive and should be in English.)

**Answer:** /dev

## 2. device types

**Question:** What is the symbol for character devices in the `ls -l` command? (Provide the single lowercase English character as your answer)

**Answer:** c

## 3. Device Names

**Question:** What would commonly be the device name for the first partition on the second SCSI disk? Please provide the answer in English, paying attention to the correct case.

**Answer:** /dev/sdb1

## 4. sysfs

**Question:** What directory is used to view detailed information on devices? Please answer in English.

**Answer:** /sys

## 5. udev

**Question:** What dynamically adds and removes devices?

**Answer:** udev

## 6. lsusb, lspci, lssci

**Question:** What command is used to view a list of connected USB devices? (Please answer in lowercase English characters only.)

**Answer:** lsusb

## 7. dd

**Question:** What is the `dd` option for block size? Please answer using only lowercase English letters.

**Answer:** bs
