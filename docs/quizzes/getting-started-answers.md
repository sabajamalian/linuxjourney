# Getting Started Quiz Answers

## 1. Linux History

**Question:** Who developed the Linux kernel? Please answer in English and be mindful of capitalization.

**Answer:** Linus Torvalds

## 2. Choosing a Linux Distribution

**Question:** What manages hardware in a Linux system? (Answer in English, paying attention to capitalization)

**Answer:** Linux Kernel

## 3. Debian

**Question:** What kind of release do Testing and Unstable have? Please answer in English and be mindful of capitalization.

**Answer:** Rolling

## 4. Red Hat Enterprise Linux

**Question:** What is the underlying package management system that Red Hat Enterprise Linux is built on? Please answer in English, using all uppercase letters for the acronym.

**Answer:** RPM

## 5. Ubuntu

**Question:** What operating system is Ubuntu based on? (Please answer in English, paying attention to capitalization.)

**Answer:** Debian

## 6. Fedora

**Question:** What is RHEL branched off of? (Please answer in English, paying attention to capitalization.)

**Answer:** Fedora

## 7. Linux Mint

**Question:** What popular distribution is Linux Mint based on? (Answer in English, case-sensitive)

**Answer:** Ubuntu

## 8. Gentoo

**Question:** What package management system does Gentoo use? (Your answer should be in English and pay attention to capitalization.)

**Answer:** Portage

## 9. Arch Linux

**Question:** What is the name of the package manager used by Arch Linux? Please answer in English, paying attention to capitalization.

**Answer:** Pacman

## 10. openSUSE

**Question:** What is the name of openSUSE's administration and installation tool? Please answer using only lowercase English letters.

**Answer:** yast
