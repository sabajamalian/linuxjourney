# DNS Quiz Answers

## 1. What is DNS?

**Question:** True or false: DNS helps us find MAC addresses for hostnames?

(Please answer in English with "True" or "False". The answer is case-sensitive.)

**Answer:** False

## 2. DNS Components

**Question:** What resource record type is used for mail exchangers?

**Answer:** MX

## 3. DNS Process

**Question:** What is the abbreviation for the nameservers where .com, .net, .org, etc., addresses are found? Please answer using only uppercase English letters.

**Answer:** TLD

## 4. /etc/hosts

**Question:** What file is used to map hostnames to IP addresses on our machines? (Please answer in English, paying attention to case sensitivity).

**Answer:** /etc/hosts

## 5. DNS Setup

**Question:** What is the de facto DNS server for Linux?

**Answer:** BIND

## 6. DNS Tools

**Question:** What tool is used to get detailed information about DNS name servers? Please answer using only lowercase English characters.

**Answer:** dig
