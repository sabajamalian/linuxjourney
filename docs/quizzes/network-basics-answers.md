# Network Basics Quiz Answers

## 1. Network Basics

**Question:** What is the local area network known as? Please answer in English, paying attention to capitalization.

**Answer:** LAN

## 2. OSI Model

**Question:** What is used as the theoretical model of networking? (Please answer in English, using the acronym in uppercase.)

**Answer:** OSI

## 3. TCP/IP Model

**Question:** What is the top layer of the TCP/IP model? (Please answer in English. Note that the answer is case-sensitive.)

**Answer:** Application

## 4. Network Addressing

**Question:** How many bytes are in an IPv4 address?

**Answer:** 4

## 5. Application Layer

**Question:** What layer is used to present the packet data in a user-friendly format? (Please answer in English and pay attention to capitalization.)

**Answer:** Application

## 6. Transport Layer

**Question:** What is a reliable transport protocol? (Your answer should be in English and is case-sensitive).

**Answer:** TCP

## 7. Network Layer

**Question:** What are the smaller networks that make up the Internet called? Please answer using a single, lowercase English word.

**Answer:** subnets

## 8. Link Layer

**Question:** What protocol is used to find the MAC address of a host on the same local network? (Please answer with the English acronym in uppercase letters).

**Answer:** ARP

## 9. DHCP Overview

**Question:** What are the four steps in the DHCP process, in order? Please answer in English, using uppercase words separated by a comma and a space.

**Answer:** DISCOVER, OFFER, REQUEST, ACK
