# Text-Fu Quiz Answers

## 1. stdout (Standard Out)

**Question:** What redirector do you use to append output to a file?

**Answer:** `>>`

## 2. stdin (Standard In)

**Question:** What operator is used to redirect stdin? Please answer with only the required symbol.

**Answer:** <

## 3. stderr (Standard Error)

**Question:** What is the operator used to redirect the `stderr` stream? Please provide the exact operator in your answer.

**Answer:** 2>

## 4. pipe and tee

**Question:** What single character represents the pipe operator in a Linux command? Please answer with only the symbol.

**Answer:** |

## 5. env (Environment)

**Question:** Which command displays all of your current environment variables? (Please answer in English, using only the lowercase command name).

**Answer:** env

## 6. cut

**Question:** What command would you use to get the first character of every line in a file?

**Answer:** cut -c 1

## 7. paste

**Question:** What flag do you use with `paste` to make everything go on one line?

**Answer:** -s

## 8. head

**Question:** What flag would you use with the `head` command to change the number of lines you want to view? Please answer using only the English flag, paying attention to case sensitivity.

**Answer:** -n

## 9. tail

**Question:** What is the flag used to follow a file in `tail`? (Please answer in English, paying attention to case sensitivity).

**Answer:** -f

## 10. expand and unexpand

**Question:** What command is used to convert tabs to spaces? (Please answer using the lowercase English command name.)

**Answer:** expand

## 11. join and split

**Question:** What command would you use to join files named `cat`, `dog`, `cow`? Please provide the full command in English. The command and filenames should be in lowercase.

**Answer:** join cat dog cow

## 12. sort

**Question:** What flag do you use to perform a reverse sort?

**Answer:** -r

## 13. tr (Translate)

**Question:** What command is used to translate characters? (Please answer in lowercase English letters only)

**Answer:** tr

## 14. uniq (Unique)

**Question:** What command would you use to remove adjacent duplicate lines in a file? Please answer using only the command name in lowercase English letters.

**Answer:** uniq

## 15. wc and nl

**Question:** Which command and option would you use to display only the total word count of a file? Please answer using only the command and its option in English. The answer is case-sensitive.

**Answer:** wc -w

## 16. grep

**Question:** What command do you use to find a certain pattern in a file? Please answer in a single lowercase English word.

**Answer:** grep
