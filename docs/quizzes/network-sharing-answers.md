# Network Sharing Quiz Answers

## 1. File Sharing Overview

**Question:** What command can you use to securely copy files from one host to another? Please answer with the command name only, in lowercase English letters.

**Answer:** scp

## 2. rsync

**Question:** What command, known for its delta-transfer algorithm, is particularly useful for creating efficient data backups? Please answer in English, paying attention to case sensitivity.

**Answer:** rsync

## 3. Simple HTTP Server

**Question:** For Python 3, what is the name of the module used to create a simple HTTP server? (Please answer in English, paying attention to case sensitivity).

**Answer:** http.server

## 4. NFS

**Question:** What tool is used to manage mount points automatically? Please answer in English, and note that the answer is case-sensitive.

**Answer:** automount

## 5. Samba

**Question:** What is the name of the protocol, an early dialect of SMB, that was developed for file sharing? Please answer in English, paying attention to capitalization.

**Answer:** CIFS
