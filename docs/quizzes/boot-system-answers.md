# Boot System Quiz Answers

## 1. Boot Process Overview

**Question:** What is the last stage in the Linux boot process? (Please answer in English, paying attention to case sensitivity).

**Answer:** init

## 2. Boot Process: BIOS

**Question:** What does the BIOS load? Please answer in a single word, in English, and in lowercase.

**Answer:** bootloader

## 3. Boot Process: Bootloader

**Question:** What kernel parameter makes it so you don't see bootup messages? Please answer with the single-word parameter in lowercase English.

**Answer:** quiet

## 4. Boot Process: Kernel

**Question:** What is used in modern systems to load a temporary root filesystem? Please answer in English, using only lowercase letters.

**Answer:** initramfs

## 5. Boot Process: Init

**Question:** What is the newest standard for init? (Please answer in lowercase English letters only)

**Answer:** systemd
