# Command Line Quiz Answers

## 1. The Shell

**Question:** What is the exact output to the display when you type `echo Hello World`? Please answer in English, paying close attention to capitalization and spacing.

**Answer:** Hello World

## 2. pwd (Print Working Directory)

**Question:** Which command is used to find the directory you are currently in? (Please answer in English, using only the command name in lowercase.)

**Answer:** pwd

## 3. cd (Change Directory)

**Question:** If you are in `/home/pete/Pictures` and want to navigate to the parent directory (`/home/pete`), what is the full command you should use? Please answer in English, paying attention to case and spacing.

**Answer:** cd ..

## 4. ls (List Directories)

**Question:** What command would you use to see hidden files? Please answer in English, paying attention to case sensitivity.

**Answer:** ls -a

## 5. touch

**Question:** How do you create a file called `myfile`? Please answer using only the English command, paying attention to case sensitivity.

**Answer:** touch myfile

## 6. file

**Question:** What command can you use to find the file type of a file?

**Answer:** file

## 7. cat

**Question:** What command is used to display the contents of a file on the command line? (Note: Your answer should be a single, lowercase English word.)

**Answer:** cat

## 8. less

**Question:** How do you quit out of the `less` command? Please provide the single character key as your answer. Note: the answer is a case-sensitive English letter.

**Answer:** q

## 9. history

**Question:** What is the command to clear the terminal? (Please answer in lowercase English letters only)

**Answer:** clear

## 10. cp (Copy)

**Question:** What flag do you need to specify to copy over a directory?

**Answer:** -r

## 11. mv (Move)

**Question:** Using the `mv` command, how would you rename a file named `cat` to `dog`? Please provide the full command. Note: The answer is case-sensitive and should be entered in lowercase English.

**Answer:** mv cat dog

## 12. mkdir (Make Directory)

**Question:** What command is used to make a directory? Please answer using only the lowercase English command.

**Answer:** mkdir

## 13. rm (Remove)

**Question:** How do you remove a file named `myfile`? Your answer must be in English and use the exact, case-sensitive command.

**Answer:** rm myfile

## 14. find

**Question:** What option should you specify for the `find` command to search by name? Please answer using only the English option, paying attention to the required format (e.g., -option).

**Answer:** -name

## 15. help

**Question:** How do you get quick command-line help for built-in Bash commands? (Please provide the single command name in English and in lowercase.)

**Answer:** help

## 16. man

**Question:** How do you see the manual for a command? (Please answer using only the command name in lowercase English letters).

**Answer:** man

## 17. whatis

**Question:** What command can you use to see a small description of a command? Please answer in English, paying attention to lowercase.

**Answer:** whatis

## 18. alias

**Question:** What command is used to create an alias? Please answer in lowercase English.

**Answer:** alias

## 19. exit

**Question:** What is the most common command to exit from the Linux shell? Please answer using only a single lowercase English word.

**Answer:** exit
