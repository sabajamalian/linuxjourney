# Subnetting Quiz Answers

## 1. IPv4

**Question:** How many bytes are in an IPv4 address?

**Answer:** 4

## 2. Subnets

**Question:** A subnet is defined by a network prefix and a subnet mask. True or False? (Please answer with 'True' or 'False'. The answer is case-sensitive and must be in English.)

**Answer:** True

## 3. Subnet Math

**Question:** What is the binary equivalent of 255?

**Answer:** 11111111

## 4. Subnetting Cheats

**Question:** What is the binary conversion of 123? Please provide the answer in English characters (numbers).

**Answer:** 01111011

## 5. CIDR

**Question:** How many bits does an IPv4 address consist of?

**Answer:** 32

## 6. NAT

**Question:** What is used to represent a single private address to the internet?

**Answer:** NAT

## 7. IPv6

**Question:** What is the name of the IP protocol designed to increase the number of available addresses for hosts on the Internet? Please answer in English using the protocol's common name, paying attention to capitalization.

**Answer:** IPv6
