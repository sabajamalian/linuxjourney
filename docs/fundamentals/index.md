# Fundamentals

Welcome to the Fundamentals learning path! This is where your Linux journey begins.

## Topics

| Topic | Description |
|:------|:------------|
| [Getting Started](getting-started/) | Learn about Linux history, distributions, and how to get started |
| [Command Line](command-line/) | Master the terminal and essential commands |
| [Text-Fu](text-fu/) | Learn text manipulation commands |
| [Advanced Text-Fu](advanced-text-fu/) | Dive into text editors (Vim/Emacs) and regular expressions |
| [User Management](user-management/) | Manage users, groups, and accounts |
| [Permissions](permissions/) | Understand file permissions and ownership |
| [Processes](processes/) | Work with processes and signals |
| [Packages](packages/) | Install and manage software packages |

Start with **Getting Started** and work your way through each topic in order.
