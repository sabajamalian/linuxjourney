# Troubleshooting Quiz Answers

## 1. ICMP

**Question:** What is the ICMP type for an echo request? Please answer with the numerical value only.

**Answer:** 8

## 2. ping

**Question:** What is the roundtrip time unit of measurement? Please answer in English, paying attention to case sensitivity.

**Answer:** ms

## 3. traceroute

**Question:** What gets decremented by one when making hops across the network? (Please answer in English, paying attention to capitalization.)

**Answer:** TTL

## 4. netstat

**Question:** What port is used for HTTPS?

**Answer:** 443

## 5. Packet Analysis

**Question:** What is the flag to capture a specific interface with tcpdump? Please answer using only the required flag in English. The answer is case-sensitive.

**Answer:** -i
