# Network Configuration Quiz Answers

## 1. Network Interfaces

**Question:** What is the legacy command used to configure a Linux network interface? Please answer in English, using only lowercase letters.

**Answer:** ifconfig

## 2. route

**Question:** When using the legacy `route` command, what is the flag used to delete a route? Please answer in English, paying attention to case.

**Answer:** del

## 3. dhclient

**Question:** What tries to assign IP addresses with the DHCP protocol?

**Answer:** dhclient

## 4. Network Manager

**Question:** What is the command to view a summary of NetworkManager's state and devices as shown in the lesson? Please answer using only the English command name in lowercase.

**Answer:** nm-tool

## 5. arp

**Question:** What command can you use to view your ARP cache?

**Answer:** arp
