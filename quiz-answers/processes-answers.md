# Processes Quiz Answers

## 1. ps (Processes)

**Question:** What `ps` flag, when used with the `a` and `x` flags, is used to view detailed, user-oriented information about processes? Please answer with a single lowercase English letter.

**Answer:** u

## 2. Controlling Terminal

**Question:** What value is given for a process that does not have a controlling terminal?

**Answer:** ?

## 3. Process Details

**Question:** What manages and controls all Linux processes? Please answer in a single English word, all lowercase.

**Answer:** kernel

## 4. Process Creation

**Question:** What system call creates a new process? (Please answer in a single lowercase English word.)

**Answer:** fork

## 5. Process Termination

**Question:** What is the most common termination status for a process succeeding?

**Answer:** 0

## 6. Signals

**Question:** What signal is unblockable? Please answer in English, using the exact signal name and paying attention to capitalization.

**Answer:** SIGKILL

## 7. kill (Terminate)

**Question:** What is the signal name for the default `kill` command? Please answer in English. Note that the answer is case-sensitive.

**Answer:** SIGTERM

## 8. niceness

**Question:** If you want a process to get more CPU priority, should you use a lower or higher nice number? Please answer in a single English word, all lowercase.

**Answer:** lower

## 9. Process States

**Question:** What STAT code is used to represent an uninterruptible sleep? (Please provide the single, uppercase English letter for the state code.)

**Answer:** D

## 10. /proc filesystem

**Question:** What virtual filesystem stores process information? Please answer in English, paying attention to case sensitivity.

**Answer:** /proc

## 11. Job Control

**Question:** What command is used to list background jobs? (Please answer in English, using only lowercase letters.)

**Answer:** jobs
