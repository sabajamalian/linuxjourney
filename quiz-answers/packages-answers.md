# Packages Quiz Answers

## 1. Software Distribution

**Question:** What package format is used by Ubuntu and Debian?

**Answer:** deb

## 2. Package Repositories

**Question:** On a traditional Debian system, what is the full path to the main file that lists package repositories? Please answer using the full file path.

**Answer:** /etc/apt/sources.list

## 3. tar and gzip

**Question:** What tar flag is used to create archives? Please answer with a single lowercase English letter.

**Answer:** c

## 4. Package Dependencies

**Question:** What is a collection of pre-compiled code that multiple programs can use? (Please answer in English, paying attention to uppercase and lowercase letters).

**Answer:** Libraries

## 5. rpm and dpkg

**Question:** What is the package management tool for `.deb` files?

**Answer:** dpkg

## 6. yum and apt

**Question:** What command is used to show package information on a Debian system? Please answer in English, paying attention to case sensitivity.

**Answer:** apt show

## 7. Compile Source Code

**Question:** What should you use instead of `make install` ALWAYS? (Please answer in English, paying attention to case sensitivity).

**Answer:** checkinstall
