# Logging Quiz Answers

## 1. System Logging

**Question:** What is the daemon that manages logs on newer Linux systems? (Please answer in English, paying attention to case sensitivity).

**Answer:** rsyslogd

## 2. syslog

**Question:** What command can you use to manually log a message?

**Answer:** logger

## 3. General Logging

**Question:** Which log file typically records everything except authentication messages? (Please answer in English, using only lowercase letters.)

**Answer:** syslog

## 4. Kernel Logging

**Question:** What command can be used to view kernel bootup messages? Please answer using only the lowercase English command.

**Answer:** dmesg

## 5. Authentication Logging

**Question:** On Debian-based systems, what is the name of the log file used for user authentication? Please answer using the filename only. The answer is case-sensitive.

**Answer:** auth.log

## 6. Managing Log Files

**Question:** What is the primary utility used for log rotation and managing Linux logs? Please answer in lowercase English.

**Answer:** logrotate
