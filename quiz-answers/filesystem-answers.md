# Filesystem Quiz Answers

## 1. Filesystem Hierarchy

**Question:** What directory is used to store logs? (Please provide the full path. The answer is case-sensitive and must be in English.)

**Answer:** /var

## 2. Filesystem Types

**Question:** What is the most common and default filesystem type for many Linux distributions? (Please answer in English, paying attention to case sensitivity).

**Answer:** ext4

## 3. Anatomy of a Disk

**Question:** What partition type is used to create more than 4 partitions in the MBR partitioning scheme? (Please answer in a single lowercase English word.)

**Answer:** extended

## 4. Disk Partitioning

**Question:** What is the `parted` command to make a partition? (Please answer in English, paying attention to case sensitivity).

**Answer:** mkpart

## 5. Creating Filesystems

**Question:** What command is used to create a filesystem? Please answer in English.

**Answer:** mkfs

## 6. mount and umount

**Question:** What command is used to attach a filesystem? (Please use a single, lowercase English word for your answer.)

**Answer:** mount

## 7. /etc/fstab

**Question:** What file is used to define how filesystems should be mounted? (Please provide the full path. The answer is case-sensitive.)

**Answer:** /etc/fstab

## 8. swap

**Question:** What is the command to enable swap space on a device?

**Answer:** swapon

## 9. Disk Usage

**Question:** What command is used to show how much space is free on your disk? Please answer in lowercase English letters.

**Answer:** df

## 10. Filesystem Repair

**Question:** What command is used to check the integrity of a filesystem?

**Answer:** fsck

## 11. Inodes

**Question:** How do you see how many inodes are left on your system? (Please answer in English, paying attention to case sensitivity.)

**Answer:** df -i

## 12. symlinks

**Question:** What is the command and its primary option used to create a symlink? Your answer must be in English and is case-sensitive. Please include the space between the command and the option.

**Answer:** ln -s
