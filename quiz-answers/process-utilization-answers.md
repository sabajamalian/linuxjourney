# Process Utilization Quiz Answers

## 1. Tracking processes: top

**Question:** What command displays the same output as the first line in `top`? Please answer using only the lowercase English command name.

**Answer:** uptime

## 2. lsof and fuser

**Question:** What command is used to list open files and their associated process information? (Please answer in English, using only lowercase letters.)

**Answer:** lsof

## 3. Process Threads

**Question:** True or false, all processes start out single-threaded.

**Answer:** True

## 4. CPU Monitoring

**Question:** What command can you use to see the system's load average? Please answer in English, and note that the command is case-sensitive.

**Answer:** uptime

## 5. I/O Monitoring

**Question:** What command can be used to view I/O and CPU usage? (Please answer in lowercase English characters only)

**Answer:** iostat

## 6. Memory Monitoring

**Question:** What tool is used to view memory utilization? (Please answer in English, paying attention to case sensitivity).

**Answer:** vmstat

## 7. Continuous Monitoring

**Question:** What is a good tool to use for monitoring system resources?

**Answer:** sar

## 8. Cron Jobs

**Question:** What is the command to edit your personal cron jobs? (Please answer using the exact lowercase command and its option.)

**Answer:** crontab -e
