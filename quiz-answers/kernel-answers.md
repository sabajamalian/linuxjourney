# Kernel Quiz Answers

## 1. Overview of the Kernel

**Question:** What level of the operating system manages devices? (Please answer in a single, lowercase English word.)

**Answer:** kernel

## 2. Privilege Levels

**Question:** What ring number has the highest privileges?

**Answer:** 0

## 3. System Calls

**Question:** What is used to switch from user mode to kernel mode? Please answer in English, using two words.

**Answer:** System call

## 4. Kernel Installation

**Question:** How do you see the kernel version of your system?

**Answer:** uname -r

## 5. Kernel Location

**Question:** In the `/boot` directory, what is the typical name for the compressed Linux kernel image file? Please answer in English, paying attention to case sensitivity.

**Answer:** vmlinuz

## 6. Kernel Modules

**Question:** What command is used to unload a module?

**Answer:** modprobe -r
