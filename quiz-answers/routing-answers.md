# Routing Quiz Answers

## 1. What is a router?

**Question:** How do packets measure distance? (Please answer in English. The answer is case-sensitive.)

**Answer:** Hops

## 2. Routing Table

**Question:** If a destination is not found in the routing table, where are the packets sent? Please answer with a single English word, paying attention to capitalization.

**Answer:** Gateway

## 3. Path of a Packet

**Question:** Which protocol is used to find the MAC address of a host on the local network, given its IP address? Please answer with the three-letter acronym in all uppercase.

**Answer:** ARP

## 4. Routing Protocols

**Question:** What is the term for the state where all routing tables on a network agree on the network topology? (Please answer in English, paying attention to capitalization.)

**Answer:** Convergence

## 5. Distance Vector Protocols

**Question:** True or false: Distance vector protocols use the route with the least amount of bandwidth?

**Answer:** False

## 6. Link State Protocols

**Question:** What is one of the most common link state protocols?

**Answer:** OSPF

## 7. Border Gateway Protocol

**Question:** What protocol basically makes the internet work? Please answer using the uppercase English acronym.

**Answer:** BGP
