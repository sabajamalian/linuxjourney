# Welcome to Linux Journey

A comprehensive, beginner-friendly guide to learning Linux.

---

## Learning Paths

Linux Journey is organized into three progressive learning paths:

### [Fundamentals](docs/fundamentals/)
**Start here if you're new to Linux.** Learn the basics of the command line, file manipulation, text processing, user management, permissions, processes, and package management.

Topics covered:
- Getting Started
- Command Line
- Text-Fu
- Advanced Text-Fu
- User Management
- Permissions
- Processes
- Packages

### [Administration](docs/administration/)
**Level up your Linux skills.** Dive into system administration topics including devices, filesystems, boot process, kernel, init systems, process utilization, and logging.

Topics covered:
- Devices
- Filesystem
- Boot System
- Kernel
- Init
- Process Utilization
- Logging

### [Networking](docs/networking/)
**Master Linux networking.** Learn network fundamentals, configuration, sharing, subnetting, routing, DNS, and troubleshooting.

Topics covered:
- Network Basics
- Network Configuration
- Network Sharing
- Subnetting
- Routing
- DNS
- Troubleshooting
